Documentação do Ficheiro GerarAnual.ps1

Descrição

O ficheiro GerarAnual.ps1 é um script em PowerShell utilizado para gerar os ficheiros de Excel necessários para a gestão das atividades dos alunos ao longo do ano letivo. Este script cria pastas para cada mês e gera os respetivos ficheiros de CAF, Dança e Lanche com a estrutura correta.

Pré-requisitos

Pré-requisito Descrição
PowerShell Deve estar instalado (normalmente já vem com o Windows).
Módulo ImportExcel Permite a manipulação de ficheiros Excel. Deve ser instalado através do PowerShell com o comando: 
Install-Module -Name ImportExcel -Scope CurrentUser
Ficheiros de Entrada O script requer um ficheiro alunos.csv que contém a lista de alunos e um ficheiro precos.csv com as informações de preços.

Funcionamento do Script

O script realiza as seguintes operações:

  1. Importa os dados dos alunos a partir do ficheiro alunos.csv.
  2. Verifica se o ficheiro contém a coluna Nome.
  3. Criar um diretório para cada mês do ano letivo, de setembro a julho.
  4. Gera três tipos de ficheiros para cada mês:
    • CAF.xlsx: Contém os alunos, os seus identificadores e os dias do mês.
    • Danca.xlsx: Contém apenas os alunos e a informação sobre se frequentam ou não as aulas de dança.
    • Lanche.xlsx: Similar ao ficheiro de dança, mas para o lanche.
  5. Exporta os dados para os respetivos ficheiros Excel, organizados por mês.

Estrutura do Ficheiro de Entrada

Coluna Descrição
Nome Nome do aluno.
Contribuinte Número de identificação do contribuinte do aluno.

O ficheiro alunos.csv deve ter a seguinte estrutura:

Nome Contribuinte
João Silva 500234123
Maria Oliveira 500234124
... ...

Execução do Script

Para executar o script, siga os seguintes passos:

  1. Abra o PowerShell.
  2. Navegue até a diretoria onde está localizado o ficheiro GerarAnual.ps1:
    3. cd caminho/para/a/diretoria
  3. Execute o script com o seguinte comando:
    4. .\GerarAnual.ps1

Após a execução, pastas para cada mês e os respetivos ficheiros de Excel serão gerados na diretoria atual.

Conclusão

Este script permite a geração automática dos ficheiros necessários para gerir as atividades dos alunos, facilitando a organização e o acompanhamento das atividades ao longo do ano letivo.
