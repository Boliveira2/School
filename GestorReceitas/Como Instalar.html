Tutorial de Instalação do Gerador de Relatório Mensal

Requisitos

Antes de começar, você precisará de:

  • Python: Versão 3.6 ou superior
  • pip: Gerenciador de pacotes para Python
  • PowerShell: Para executar o script GerarAnual.ps1
  • Editor de Código: Qualquer editor de código de sua escolha (Visual Studio Code, PyCharm, etc.)
  • Ficheiros de Entrada: alunos.csv, precos.csv, CAF.xlsx

Passo 1: Instalar Python

  1. Download do Python:

    Acesse o site oficial do Python e baixe a versão mais recente do Python (3.6 ou superior). Você pode baixar diretamente clicando aqui para Windows ou aqui para macOS.

  2. Instalação:

    Execute o instalador e certifique-se de marcar a opção "Add Python to PATH" durante a instalação. Complete a instalação seguindo as instruções na tela.

Passo 2: Verificar a Instalação do Python

Após a instalação, verifique se o Python e o pip estão instalados corretamente:

python --version
pip --version

Você deve ver a versão instalada do Python e do pip.

Passo 3: Criar um Ambiente Virtual (Opcional)

Um ambiente virtual ajuda a manter as dependências do seu projeto isoladas de outros projetos Python.

  1. No terminal, navegue até a pasta onde você deseja criar o projeto: 
    cd caminho/para/seu/projeto
  2. Crie um ambiente virtual: 
    python -m venv env
  3. Ative o ambiente virtual:
    • Windows: 
      .\env\Scripts\activate
    • macOS/Linux: 
      source env/bin/activate

Passo 4: Instalar as Bibliotecas Necessárias

Instale as bibliotecas necessárias usando o pip. No terminal, execute:

pip install pandas openpyxl

Bibliotecas Explicadas

  • pandas: Biblioteca poderosa para manipulação de dados.
  • openpyxl: Biblioteca para ler e escrever ficheiros Excel (.xlsx).

Passo 5: Instalar o PowerShell e o Módulo ImportExcel

  1. PowerShell:

    O PowerShell geralmente já está instalado no Windows. Para verificar, procure por "PowerShell" no menu iniciar.

  2. Instalar o Módulo ImportExcel:

    Abra o PowerShell como administrador e execute o seguinte comando:

    Install-Module -Name ImportExcel -Scope CurrentUser

Passo 6: Criar os Ficheiros de Entrada Usando o Script

Utilize o script GerarAnual.ps1 que deve estar na mesma diretoria onde estão os ficheiros alunos.csv e precos.csv para criar os ficheiros de entrada. Este script irá gerar os ficheiros de Excel necessários para cada mês do ano letivo.

Passo 7: Execução do Script

Para executar o script, abra o PowerShell na diretoria correta e execute:

.\GerarAnual.ps1

Após a execução, serão criados diretórios para cada mês e os respetivos ficheiros de Excel dentro desses diretórios.

Conclusão

Parabéns! Você configurou com sucesso o Gerador de Relatório Mensal. Agora você pode usar os ficheiros gerados para gerir as atividades dos alunos de forma eficaz.
